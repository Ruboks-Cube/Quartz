Used to construct [[Python Dictionaries\|Dictionaries]]
```python
thisdict = dict(name = "John", age = 36, country = "Norway")  
print(thisdict)
```
Similar to [[Using list constructor to create a list\|list()]], and [[Tuple constructor\|tuple()]]

To [[copy dictionaries]] we do this:
```python
dict2 = dict1.dict()
```
- or we use the [[copy()]] method
