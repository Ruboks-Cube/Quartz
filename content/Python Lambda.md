In [[Python]] you can create lambda functions → this is the only built-in way to create an [[anonymous function]] in python
	These are 1 line functions that accept any number of [[parameters in python\|arguments]] (so it’s like [[Arbitrary Arguments python\|Arbitrary]]) but only have one [[expression in coding\|expression]]
syntax:
```python
lambda _arguments_ : _expression_
```

So here is an example:
```python
x = lambda a,b : a+b+10 # any number of arguments is possible
print(x)
```
- When you assign the [[python variables\|variable]] that’s basically making the function the variable
