#math/geometry , [[Trigonometry\|Trigonometric Functions]], [[Triangle]]

- We know the normal [[Trigonometry\|Trigonometric Functions]], the [[Domain]] to [[Range]] ones.
	- Now lets find the [[Inverse Functions]] of those, what if we go from the [[Ratio]] to the [[Angles\|angle?]]
	- Such [[function\|functions]] are called arc functions and you find them on a calculator

| Function         | [[Inverse Functions\|Inverse]] |
| ---------------- | ------------------------------ |
| [[sine\|Sin]]    | ArcSin                         |
| [[cosine\|Cos]]  | ArcCos                         |
| [[Tangent\|Tan]] | ArcTan                         |
- These functions can be expressed line $y=\sin^-1$ which means the [[Inverse Functions]] 
	- On a calculator the inverse buttons may be marked as $\arcsin$, $asin$, or $\sin^{-1}$
	- This form " $\sin^{-1}$ " can be misleading because raising it to the [[Negative Exponents\|Negative power]] implies the [[Reciprocal]]. 
- Using the different functions:
	![[Files/Images/Pasted image 20240713171426.png|Diagram|200]]
	- Using Arcsin
		- The [[sine]] is just opposite over [[Hypotenuse]]. In the above figure $\sin C= \frac{6}{10}=0.6$. The sin of $C$ is known so we use the inverse function to find the [[Angles\|angle]]. 
			- $C=\arcsin(0.6)$. This basically says, "C is the angle whose sin is 0.6"
			- Using a calculator we find $\arcsin(0.6)=36.86^{\circ}$ so angle $C$ which has a [[sine]] of $0.6$ has a measure of $36.86^{\circ}$ 
	- Solving a problem
		![[Files/Images/Pasted image 20240714084006.png|Diagram|200]]
		- In the diagram you can see $\sin 40^{\circ} = \frac{64.28}{100}=0.6428$. 
			- If [[Angles\|Angle]] $\angle B$ is not given but you know the [[Line Segment\|side lengths]] of the [[Triangle]] then you can find $\angle B$ from the [[Inverse Functions\|Inverse Function]] ${} y=\sin^{-1}(0.6248) {}$. Using a calculator you can see that $y=\sin^{-1}(0.6428) \approx 40.00093^{\circ}$ or $40^{\circ}$.
			- Looking at [[Angles\|Angle]] $\angle A$ it follows that $\cos A=0.6428$ and the angle [[Complementary Angles\|Complementary]] to $\angle B$ $m\angle A=50^{\circ}$
	- [[checking your answer]]
		- You can check if you got it right by [[Triangle Sum theorem\|checking if the angles add up to 180]]. 
	- Finding Side lengths
		- You can find side lengths with the [[Trigonometry]]
			- If you know a trigonometric ratios and need to find the other side lengths you can do so without the [[Pythagorean Theorem]]
			- If you want to solve for side length $x$ and you know that $\sin A=\frac{x}{120}$ you can solve for $x$ just like it's a normal [[Linear Equation]]. SIN A IS JUST A [[variable math\|variable]], it's just a value you can find on a calculator!!!
