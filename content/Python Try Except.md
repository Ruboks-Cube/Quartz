In [[Python]] the [[Try in python\|try]] lets you check for [[python errors]] and [[python except\|except]] lets you deal with them
- Without the try block, the program will just crash when an error is raised

```python
try:  
  print(x)  # x isn't defined so there is an error
except:  
  print("An exception occurred")
```
- If you think an error happens you’ll use [[Try in python\|try]] and an [[python except\|except]] will follow to know what will happen with the error
