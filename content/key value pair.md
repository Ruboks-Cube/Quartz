[[key in python dictionaries\|key]] [[value in dictionary\|Value]] pairs are the unit that make up a [[Python Dictionaries]]
```python
thisdict = {  
  "brand": "Ford",  
  "model": "Mustang",  
  "year": 1964  
}  
print(thisdict["brand"])
```
- In this example `"brand": "Ford",` is a [[key value pair]]

- They are also used in [[keyword arguments python]]
