#Coding/python 

Negative [[indexing]] is used to start from the *end* of a [[Sequence in coding\|sequence]]
```python
a = [1, 2, 3, 4]
print(a[-1]) #prints 4
```

Here, a is the [[Sequence\|sequence]] and we know that the [[indexing\|index]] 3 is 4, but we could access it with -1

We can use the [[Print()\|print]] [[Functions in coding\|function]] in order to print the negative [[indexing\|index]], just like a normal index.
