# Intuitive Explanation
The [[Perfect Squares\|Square]] of a [[Radicals and Square Roots\|square root]] refers to a situation like this:
$$\left(\sqrt{x}\right)^2$$
The square root of a square is this: $\sqrt {x^2}$


First, we can try some small numbers, like 4, if we square root 4 (2), then square it, (4), the answer is 4.
The same is for 9, another [[Perfect Squares\|Perfect Square]], the answer is 9 when we put it in x. So is the square of a square root just the number inside the radical?

Lets see, we know
- Squaring something means multiplying it by itself
- The root square of something is what we square to get the number inside the radical!
	- Meaning $$\sqrt{x}\cdot\sqrt{x}=x$$
	- If you don't understand, the square root is asking, what do we multiply by itself to get x? The answer would be the square root of x
- Now we know that: $$\sqrt{x}\cdot\sqrt{x}=\sqrt{x}^2=x$$
It's very similar to the term square root of a square ($\sqrt {x^2}$) because it's basically the same thing. $\sqrt x^2=(x^{1/2})^2=x^{1/2*2}=x^{2*1/2}=(x^2)^{1/2}=\sqrt {x^2}$
