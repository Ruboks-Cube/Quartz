# Example Problem
The product of 360, 200, and 81 is a cube of what integer?
Hmm, so not only do we need to multiply 360,200, *and* 81, we also need to find an integer that when cubed is the product of these numbers! Gah! This seems very hard? There should be a better solution to this, think, think!

Well, first, what do we know about this problem. It's a <mark style="background: #FFF3A3A6;">product of numbers</mark>, that is a cube of another integer. But remember, a cube is just a product of the same number over and over again. So right now, what we have is
$$x^3=360\cdot200\cdot81$$
So, it's basically the same number, but now we might realize that we can <mark style="background: #ABF7F7A6;">manipulate</mark> 360, 200, and 81 so they are the same number. Because this is [[Multiplication]] and we don't have to multiply the same numbers due to the [[Associative Property]], we can break up the numbers into [[Multiple]]s of the original! That way, we can rearrange those numbers because of the associative property into 3 identical groups so we have 3 of the same number being multiplied to itself 3 times.

## solving the problem
![[Files/Images/Prime factorization 2024-01-02 10.47.13.excalidraw]]
Using the [[Exponent Rules!]], specifically the [[Product of Powers, quotient of powers]], we get this:
$$360=2^3\cdot3^2\cdot5$$
$$200=2^3\cdot5^2$$
$$81=3^4$$
Now that we have broken down the number to it's smallest possible components, [[Prime Numbers]] since once you get to prime numbers, there are no multiples left to do [[Factoring]].

Now all we have to do is rearrange and divide each exponent by 3.

![[Files/Images/Prime factorization 2024-01-02 10.54.18.excalidraw]]
We get our answer through [[Exponent Rules!]] and the [[Associative Property]].

# Prime Factorization

Prime Factorization is [[Factoring]] a number down to [[Prime Numbers]], since the primes have to [[Factor]] you can't break them down any further. Hence the name, prime factorization!
