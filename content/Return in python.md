Think about [[Returning in code]] 
	Same for [[Python]]!
When you [[Calling a function in python\|call a function]] what is returned is what the function will equal

```python
def my_function(x):  
  return 5  x  
print(my_function(3))  # without return there is nothing to print!
print(my_function(5))  
print(my_function(9))
```
- Returning basically allows the [[Python custom functions\|custom function]] to equal a value of our choice when it is called
