We can add [[tuple\|tuples]] to [[tuple\|tuples]]

```python
thistuple = ("apple", "banana", "cherry")  
y = ("orange",)  
thistuple += y  

print(thistuple)
```
[[Assignment Operators python]] 
	Used +=

You can join tuples with the + [[Python Operators\|Operator]]
```python
tuple1 = ("a", "b" , "c")  
tuple2 = (1, 2, 3)  
  
tuple3 = tuple1 + tuple2  
print(tuple3) # output ('a', 'b', 'c', 1, 2, 3)
```
