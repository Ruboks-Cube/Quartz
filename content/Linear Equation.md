# What is a linear equation
Linear Equations are [[equations\|Equations]] that have the [[variable math\|variables]] only in the first exponent. They are [[Linear Expression]]s but an equation that can be turned into a [[math functions\|Function]] that is a straight-line. (Linear means straight)

- Each term only has 1 variable

## Solving Linear equations
When we have something like $$3x+15=45$$
The goal is to find x. We can find x by isolating it to one side, since 3x+15=45, we know that x is a number where 3 times x plus 15 = 45. So how do we get the value? You may realize, that plugging in values may seem difficult, but worry not! There is an approach to this!

![[Files/Images/Linear Equations 2024-01-10 20.28.16.excalidraw]]
the first thing, is to conceptualize it as a scale. 3x+15 EQUALS 45, the value must stay consistent no matter what!  

we then solve the equation by isolating x through [[Inverse Operation]]s
![[Files/Images/Linear Equations 2024-01-10 20.29.38.excalidraw]]
Think of x as a value, x = 10, then we add a bunch of stuff, like multiply the thing by 3, 3x=30, then add 15, 3x+15=45. Now look, it looks completely different, but x=10 and 3x+15=45, they are actually the same, you can simplify the whole equation through [[Inverse Operation]]s.
