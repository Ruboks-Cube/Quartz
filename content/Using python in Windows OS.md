#Computers/OS/Windows #Coding/python 

- Many [[operating system\|operating systems]] have [[Python]] installed already
- To check in [[Computers OS Windows\|windows]], go to [[command prompt]] app and do this command:
```
C:\Users\saumy>python --version
```
- This will give u the version of python
> [!summary]
> Use command prompt to access python in windows OS
- the [[command prompt]] will allow you to run python files
	- Any text editor can run it including the notes app
- To run a python file do this command:
```
C:\Users\Your Name>python <filename>
```
