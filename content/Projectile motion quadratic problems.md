#math/algebra/inequality/equations/polynomial 

- It is important to be able to describe the path of an object near the surface of the Earth when only [[Gravity]] is acting on the object
	- We can describe this with a [[quadratic equations\|Quadratic]] [[math functions\|function]], [[Standard Form quadratic]]
		$h(t)=-\frac{1}{2}gt^2+v_{o}t+h_{o}$
	- $g=\text{gravitational constant (meters = 9.8; feet = 32)}$ -> THIS ONE YOU NEED TO REMEMBER
		- It won't always be $-16x^2$ because **what if the problem is talking in [[meter\|meters]]? Then what're you gonna do huh?** 
	- The "$-\frac{1}{2}gt^2$" part is talking about the effect of [[Gravity]] (makes the object go down)
	- $v_{o}t$ = **initial velocity** of the object (if there was no gravity it'd just be a [[Linear Equation\|linear]] path up up up like what this part describes)
	- $h_{o}$ describes the height of the object at the start (y intercept)


- Examples could be:
	- A potato is shot at a velocity of 180 feet per second on a hill that is 2 feet above sea level
		$-16x^2+180x+2$
	- A baseball is pitched at a velocity of 26 **meters** per second on the ground
		$-4.9x^2+26x$ (no y intercept)
