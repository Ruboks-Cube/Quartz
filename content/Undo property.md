#math/arithmetic/logarithm 

- [[Logarithm\|Logarithms]] and [[Exponents]] of the SAME [[base\|BASE]] “undo” each other (cancel out to one)
	- [[base and argument equals same in log]]
	- Now if we have something like $7^{\log_{7}4}=x$, how do we solve for $x$?
	- We put it in a $\log$ because its an [[Exponents\|EXPONENT!!!]] → [[exponents can be converted into logs and vice versa]]
	- So we know that $\log_{7}x=\log_{7}4$ because we converted $7^{\log_{7}4}=x$ into the logarithm version by thinking “exponent from 7 to x EQULS $\log_{7}4$” AKA $\log_{7}x=\log_{7}4$
- We can put it all into a rule like $a^{\log_{a}b}=x$
	- $x=b$ because $\log_{a}x=\log_{a}b$ and since its the same base we have $x=b$
	- Its similar to how for [[solving exponential equations]] we can [[solving exponential equations by getting them to have the same base\|solve exponential equations by getting them to have the same base]]
