### Rule Way
Using the [[Same Base exponent rules#Same base quotient of powers\|Quotient of powers same base]], we can prove how Negative exponents work as well.

Lets consider an expression like $\frac{x^2}{x^3}$, this expression is equal to $\frac{1}{x}$ because it is also $\frac{{x*x}}{x*x*x}$  so they cancel out and make $\frac{1}{x}$. But using the [[Same Base exponent rules#Same base quotient of powers\|Quotient of powers]] rule we have $x^-1$, showing that $x^{-1}=\frac{1}{x}$

Again showing that for negative exponents you take the [[Reciprocal]]
# Key ideas
- Negative exponents are when the exponent is a negative number, we take the [[Reciprocal]] to get the solution.

- for $\frac{1}{a^{-b}}$ for where $a$ and $b$ are both positive real numbers we have $\frac{1}{\frac{1}{a^b}}$ which is $a^b$.
