# What if I have to find the LCM of big numbers?
When you have big numbers, it can be annoying to find the LCM since they most likely have big prime factorizations.

But there is a pattern we can utilize to make it easier for us with [[Factoring]].
![[Files/Images/Least Common Multiple slick trick 2024-01-06 08.22.14.excalidraw]]
Here, we can see that by multiplying all numbers in a LCM problem by 2, it increases the value of the LCM by 2 also, this makes sense since we are increasing the exponent by 2. 

We can do this backwards too! When finding the LCM of something like 12,16, we can actually <mark class="hltr-red">factor *out* a 2.</mark> 
So, is this only for 2? Nope, the nice thing is, we can factor out anything! Preferably prime numbers, but when you factor a multiple of something, you can break it down! Lets try it.
