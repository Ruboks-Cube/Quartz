#math/geometry/creating , [[Creating Geometric Constructions]], [[parallel]]

# How to
- Before:                      ![[Files/Images/Pasted image 20240708173122.png]]
- Draw a line through $R$ and $PQ$ at an [[Angles\|Angle]] creating point $J$ 
- With the compass point at about half $JR$ create an arc from $J$ that [[Intersect\|intersects]] $PQ$ ![[Files/Images/Pasted image 20240708173311.png]]
- Using the same [[compass]] width go to R and create a similar [[arc]] 
- Set the compass to the lower arc                                  ![[Files/Images/Pasted image 20240708173359.png]]
- Move the compass to the upper arc and draw another arc that [[Intersect\|intersects]] the previous arc, where they intersect is point $s$
- Draw a [[line]] through $R$ and $S$                            ![[Files/Images/Pasted image 20240708173507.png]]


# Proof
![[Files/Images/Pasted image 20240708173858.png]]
1. Line segments AR and BJ are congruent -> Drawn with same compass width 
2. Line segments RS and JC are congruent -> drawn with same compass width 
3. Line segments AS and BC are congruent -> drawn with same compass width
4. [[Triangle\|Triangles]] $\triangle ARS$ and $\triangle BJC$ are [[congruent]] -> [[SSS side side side]]
5. $\angle ARS$, $\angle BJC$ are [[congruent]] -> [[corresponding parts]] of [[congruent]] [[Triangle\|Triangles]] are congruent
6. Line $AJ$ is [[Angles and parallel lines\|Transversal]] ->The angles are congruent and it's a straight line
7. Lines $RS$ and $PQ$ are parallel-> $\angle ARS$ and $\angle BJC$ are [[corresponding angles]] that are equal in measure only if lines RS and PQ are parallel
