#Coding/python 

see: [[for loop]]
- In python you can loop through any [[collections in python\|collection]] you want

basic [[syntax]]:
```python
for variable in sequence:
    # code to execute
```
- What this does is that it executes code for every value in a [[Sequence]]
- [[loop variable]]
	- [[Multiple loop variables]]
Basic examples:
```python
# loop through a list
fruits = ["apple", "banana", "cherry"]  
for x in fruits:  
  print(x)
  
# loop through a string
for x in "banana":  
  print(x)
```
