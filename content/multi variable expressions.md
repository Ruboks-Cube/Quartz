#### Introduction to distributing in multivariable expressions
![[Files/Images/Pasted image 20240422175332.png]]
- (a) $2x+4y$  each town has a team with baseball (2) and football (4), with x in baseball and y in football
- (b) now we include every town, we just have 7 of the expression in part a, so we have $7(2x+4y)$ 
- (c) this would be $7(2x)$ which simplifies to $14x$
- (d) this would be $7(4y)$ which simplifies to ${} 28y {}$
- This would be $14x+28y$, you can [[Factoring\|factor]] out the 7 in both expressions to get $7(2x+4y)$, which is the expression we got in (b)

### [[Fraction\|fractions]] with multivariable expressions
#### An introduction
![[Files/Images/Pasted image 20240422192820.png]]
- $\frac{{2x+4y}}{8}*{\frac{{3xy}}{x^2+2xy}}$
- we can combine the fractions to get $\frac{{(2x+4y)(3xy)}}{8(x^2+2xy)}$
- Expand both the numerator and denominator to get $\frac{{6x^2y+12xy^2}}{8x^2+16xy}$
	- We can factor out ${} 6xy$ from the numerator, and ${} 8x$ from denominator to get $\frac{{6xy(x+2y)}}{8x(x+2y)}$
	- the $x+2y$ on the numerator and denominator both cancel out so we have $\frac{{6xy}}{8x}$ 
- simplify further by cancelling out the $x$'s, then cancel out the $2$'s to get $\frac{3y}{4}$



#### Adding them
![[Files/Images/Pasted image 20240424200232.png]]
- $\frac{2}{r}+\frac{3}{s}=\frac{{2s}}{rs}+\frac{{3r}}{rs}$, we find the [[adding fractions with different denominators\|We find the common denominator]] which is $rs$ and we multiply both numerators by the corresponding variable so the fraction states [[equivalent]]
- Combine the fraction to get $\frac{{2s+3r}}{rs}$

##### More advanced Adding fractions multi variable
![[Files/Images/Pasted image 20240424200612.png]]
- This time we're subtracting, first find the common denominator
	- We can find a common denominator normally or by finding a simple one by considering the denominators
	- We first consider the constants, $3*2$ is $6$, so we just use 6 as our constant
	- One fraction has $x^2$, and the other one has $x$, so we just have $x^2$ in our common denominator
	- We also need a $y$, in our common denominator so we have $6x^2y$ as our common denominator
- Multiply $\frac{5y}{6x^2}$ by $y$ to get $\frac{5y^2}{6x^2y}$ and $-\frac{4}{3xy}$ by $2x$ to get $-\frac{4(2x)}{6x^2y}$
- we now have $\frac{5y^2}{6x^2y}-\frac{8x}{6x^2y}$
- Combine the fraction now that we have a common denominator to get $\frac{{5y^2-8x}}{6x^2y}$

> [!Tip] Key Idea here
>Think about your common denominator more in order to simplify it quicker, in multi variable expressions, think of the constant, and variable terms differently so we have a process instead of just multiplying them


#### Adding them but... WHAT
![[Files/Images/Pasted image 20240424202919.png]]
- $\frac{2a^3}{a^3b}+\frac{3b}{a-1}-\frac{{3b-3}}{6ab-6a}$ 
- Before we start adding, we should first see if we can [[Simplifying Fractions\|Simplify]] any of the fractions.
	- We can simplify $\frac{{2a^3}}{a^3b}$ to $\frac{2}{b}$
	- Factor out $3$ from the numerator and denominator to get $\frac{3(b-1)}{3(2ab-2a)}$ and get $\frac{{b-1}}{2ab-2a}$ as our fraction, we can also factor out $2a$ from the denominator to get $\frac{{b-1}}{2a(b-1)}$. so we have $\frac{1}{2a}$ as our fraction now
- Our expression is now $\frac{2}{b}+\frac{3b}{a-1}-\frac{1}{2a}$ 
- We try to find a common denominator, first look at constants, we need a 2, an a-1, and a $b$, so we have $2ab(a-1)$ as our common denominator

- Multiply numerator and denominator of $\frac{2}{b}$ by $2a(a-1)$ to get $\frac{4a(a-1)}{b(a-1)2a}$  which is just $\frac{4a(a-1)}{2ab(a-1)}$ 

- we multiply numerator and denominator of $\frac{3b}{a-1}$ by  $2ba$ to get $\frac{3b(2ba)}{(a-1)(2ba)}$
- This is $\frac{3b(2ba)}{2ab(a-1)}$ 

- Multiply numerator and denominator by $b(a-1)$ to get $\frac{b(a-1)}{2ab(a-1)}$ 

Now we have $\frac{{4a(a-1)+3b(2ba)-b(a-1)}}{2ab(a-1)}$ 

- Expand EVERYTHING to get $\frac{{4a^2-4a+6ab^2-ab+b}}{2ab(a-1)}$
We cannot simplify further, so we're done :)
