Just like for [[positional arguments]], you can specify if you want only [[keyword arguments python]] 
**	For keyword-only arguments you add `*,` *before* all the [[parameters in python\|arguments]]
		This is like adding a [[parameters in python\|parameter]] that’s a `*`
	Without the `*,` you can use [[positional arguments]]
