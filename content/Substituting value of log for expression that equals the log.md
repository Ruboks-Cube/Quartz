#math/arithmetic/logarithm 

- You know how [[Logarithm\|Logarithms]] can be $\log_{b}a=x$ or $b^x=a$
- BUT there’s tow ways to write each log that is useful in proving the [[Change of base logarithm rule]]
- We can do $\log_{b}a=x$ and $\log_{b}b^x$ which are both the same because ${}   b$ is $b^x=a$ and since $b^x=a$ we can [[substitution\|substitute]] it for $a$
- Additionally, since we start with a log like $\log_{b}a=x$ and we know that $b^x=a$ we can use either one for logs with different [[base\|bases]] 
	- If we CHANGE the base to $c$ we have $\log_{c}b^x=\log_{c}a$
