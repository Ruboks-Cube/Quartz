We can use [[keyword 'in' in python\|in]] and [['not In' in python\|not in]] while using a [[set in python]]
Example of using [[using print() with (in) in python\|print() with (in) keyword]] with sets:
```python
my_set = {"happy", "little", "trees"}
print("happy" in(my_set)) # output is True
```
