#math/geometry #math/geometry/transformation 

- We can write [[Transformation\|Transformations]] as [[math functions\|Functions]]. using $(x,y)$ as the [[preimage]] and using the [[image (transformation)]] 

- We're basically [[using algebra]] to write [[Geometry\|Geometric processes (transformations]] so we can express them on a [[Cartesian coordinate system\|coordinate grid]].
	- EG: We can express the [[Translation]] of a [[point]] $(x,y)$ $6$ units to the right by saying $T(x,y)=(x+6,y)$



- There's a series of functions we can write: 
	- T(x,y)=(x+a,y+b) -> [[Translation\|Translating]] by a,b
	- R(x,y)=(-x,y) -> [[Reflection\|Reflecting]] across $y$ axis
	- R(x,y)=(x,-y) -> [[Reflection\|Reflecting]] across $x$ axis
	- R(x,y)=(-y,x) -> [[Rotation\|Rotating]] $90^{\circ}$ CCW
	- R(x,y)=(-x,-y)-> [[Rotation\|Rotating]] $180^{\circ}$ CCW/CW
	- R(x,y)=(y,-x)-> [[Rotation\|Rotating]] $90^{\circ}$ $CW$
