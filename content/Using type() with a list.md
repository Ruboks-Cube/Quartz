#Coding/python 

In [[Python]] if we use [[type() function in python\|type()]] with a [[lists in python\|List]] then it returns this:
```python
mylist = [1, 2, 3, 4, 5]
print(type(mylist)) #returns <class 'list'>
```
This is for other [[Datatype\|datatypes]] as well, but instead of list it would say whatever datatype it is
[[datatypes are just built-in classes]]
