[[access object properties and methods in python]]
	We [[Accessing sub variables with the dot\|access sub variables with a dot]] in [[Python]]
When you have an [[Creating an object in python\|object]] and you want to change a [[python variables\|property]] all you have to do is access it with the dot and change the value

```python
class exampleclass:
	x = 5
	
myobject = exampleclass()
myobject.x = 6 # now for this object, x = 6
```
