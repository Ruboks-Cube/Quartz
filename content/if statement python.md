Python’s [[selection control structure\|conditional control structure]]
```python
'''if <value> <operator> <value>:
	 <if statement code>''' #notice the indent
```
- This is the basic [[syntax]] for an if statement
	- You use [[Comparison Operators python\|Comparison Operators]]
Example of a simple if statement using [[User Input]]:
>[!Example]
>```python
>name = input("What is your name:").strip().lower()
>if name = "sacha":
>	print("Hello Sacha!")
>```
- This program takes your **name**[^1], if your name is “sacha” then it greets you
