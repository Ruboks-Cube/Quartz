#math/function/exponential 


- These functions are ones where the distance of points decrease from left to right, different from [[Growth function]]
	- These happen when the $b$ value of $ab^x$ is a fraction, or the $b$ value is an [[integer]] but is raised to $^{-x}$  to we’re left with $ab^{-1*x}$ which is $a\left( \frac{1}{b} \right)^x$
	- For a decay function, you want to test the $x$ values for $0$ and $-1$ because that’s when it is “rising”
