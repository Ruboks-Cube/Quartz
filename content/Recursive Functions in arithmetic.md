#math/algebra/inequality/equations #math/algebra #math/function #math/algebra/sequence 


- Recursive Formulas give us 2 pieces of the [[math functions\|function]] 
	- The first term of the sequence
	- The pattern rule to get any term from the [[term]] before it

$$\begin{cases}a(1) = 3&\leftarrow {\text{the first term is 3}}\\\\
a(n) = a(n-1)+2&\leftarrow{\text{add 2 to the previous term}}
\end{cases}$$ 
- To solve for a specific place in the recursive formula you need to go step by step in the formula.
	- If we’re solving for $a(4)$ and we know that $a(1)=3$ then we have $a(2)=a(2-1)+2$. We have $a(1)=3$ so we have $a(2)=a(1)+2$ which means that $a(2)=5$
	- We repeat this until we get to $a(4)$
	- This here is a FORMULA way when we know that if we want to get to the 4th term in a sequence we can js go from the 3rd and add, but this is still useful to know
