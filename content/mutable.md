#Coding/python #Coding 

Mutable means can be changed

E.g, in python [[lists]] are mutable

```python
my_list = [1, 2, 3]
my_list[0] = 10      # Change first element
my_list.append(4)    # Add new element
print(my_list)       # [10, 2, 3, 4]
```

While in [[Python]], [[Strings in Python\|str]] are [[immutable]]
