[[Using del to delete part of or whole collection]]

del can be used to delete a [[variable in coding\|variable]], or part of a [[collections in python\|collection]].

Deleting a variable:
```python
x = 5
del x
print(x) # returns error as x no longer exists
```

This is the basic use of del
[[Using del to delete part of or whole collection]] is derived from this


del is different from [[remove()]] and [[pop()]] as it can delete a variable OR part of a [[collections in python\|collection]], not just an item in a list.
