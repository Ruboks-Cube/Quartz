#Coding/python #Coding 

- In other coding languages indentation is only for readability
	- In [[Python]] its an actual syntax thing
> [!success] Will work
> ```
> if 5 > 2:
> 	 print("Five is greater than two!")
> ```

> [!failure] Wont work
> ```
> if 5 > 2:
  > print("Five is greater than two!")
> ```

- The number of spaces is optional, but it must be at least 1
