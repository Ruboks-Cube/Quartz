#Coding/python 

We can loop through a [[Strings in Python\|str]] in [[Python]] using a [[for loop in python\|for loop]]
	we can loop through the [[Strings in Python\|str]] because it is a [[sequence in python\|sequence]]

```python
a = "hello world!"
for char in a:
	print(char)
```
