Elif is what you use to add on to an [[if statement python]]
	“If previous condition not true, try this one!”
```python
a = 33  
b = 33  
if b > a:  
  print("b is greater than a")  
elif a == b:  
  print("a and b are equal")
```
- Since b is not greater than a it will try the elif condition
