Access [[tuple\|tuples]] by [[indexing in python\|indexing]] just like you would a [[lists in python\|List]] 
	[[negative indexing]] works here too
	We can return a range of [[indexing\|indexes]] with [[slicing sequences in python\|slicing]] as well
```python
my_tuple = ("apple", "banana", "cherry")
print(my_tuple[1]) # Output: banana
# slicing

print(my_tuple[1:3]) # output: ('banana', 'cherry')
```
