#math/algebra #math/function #math/arithmetic 

- Composition functions are when you do [[arithmetic]] or operations with functions
- For example, you may have 2 functions $f(x)$ and $g(x)$
	- You can do the following:
		- $f(x)+g(x)$
		- $f(x)-g(x)$ 
		- $f(x)*g(x)$
		- $\frac{f(x)}{g(x)}$
		- $f(g(x))$
- You can also reverse these → Almost like [[Symmetric Property of equality]]

- I think doing composition functions are as easy as finding the value of any other [[math functions\|function]]
	- For when you have $f(g(x))$ js [[substitution\|substitute]] whatever in $g(x)$ to the $x$ of $f(x)$
- [[Domain\|finding the domain]] for composition functions is largely the same
	- If the function inside has a domain buster, that’s will not be part of the original domain
	- As you simplify always check if more domain busters happen
