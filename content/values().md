The values() [[Method]] returns a [[view in python\|view]] ([[lists in python\|List]]) of all the [[value in dictionary\|values]] in a [[Python Dictionaries\|Dictionary]]
```python
thisdict = {  
  "brand": "Ford",  
  "model": "Mustang",  
  "year": 1964  
}
x = thisdict.values()
print(x) #output: dict_values(['Ford', 'Mustang', 1964])
```

This is a [[view in python\|view]] of the dictionary just like how [[keys()]] returns a [[view in python\|view]]
