### Introduction
![[Files/Images/Pasted image 20240416195635.png]]
In this equation, we are presented with a [[Radicals and Square Roots]]. We can [[substitution\|Substitute]] $\sqrt{ x }$ for $y$ by setting $\sqrt{ x }=y$. 

Our equation is not $3y-2=30-y$. We can solve this to get $y=8$.
We can substitute our x back in now to have $\sqrt{ x }=8$
We can [[Perfect Squares\|Square]] both sides to get $x=64$.


> [!tip] Substitution
> Substitution is a very powerful tool, if you see something complicated, try to see if you can substitute it!
