The [[del keyword python]] can delete a specific [[key value pair\|Item]] in a [[Python Dictionaries\|Dictionary]]
	just specify the [[key in python dictionaries\|key]]
```python
thisdict = {  
  "brand": "Ford",  
  "model": "Mustang",  
  "year": 1964  
}  
del thisdict["model"]  
print(thisdict)
```

Additionally since [[del keyword python\|del()]] can delete an entire [[variable in coding\|variable]] it can delete the entire [[Python Dictionaries\|Dictionary]] as well
