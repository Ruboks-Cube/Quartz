#math/geometry/2d/shape/circle 

Problem:

I have a [[Circle\|Circular]] garden, the radius is 100 feet. There is 5280 feet in every mile. What is the smallest whole number of laps I must run around my garden to run at least a mile.

1. Figuring out what the Question means
	- The question means how many FULL laps do I need to make around my garden to run around a mile
		- Answer must be a whole number

2. Finding out how to solve it
	- We need to get the [[circumference]] of the circle so we can divide that by 5280
		- Get the circumference by finding the [[diameter]] and multiplying it by [[pi]]
			- get the diameter by multiplying the [[radius]] by 2

3. Follow the steps!
	- We multiply 100 by 2 to get diameter, diameter is 200
	- Circumference is 200 times pi. Approximately 628ft, since we multiply 200 by 3.14
	- How many circumferences/laps do we need to run at least a mile.
		- 10 laps is 6280ft
		- 9 laps would be 5652
		- 8 laps would be less than 5280, so we have to run at least 9 laps to run a mile
4. We run 9 laps
