# What is the rule
So what is the rule? 

The rule is that you can factor out, or add the numbers in the product of powers.

For example:
$x^n*y^n=(x*y)^n$ because $(x*y)\dots(x*y)$ n times so using the [[Commutative and Associative Property]] we have $(x*y)^n$ 

The exponent is being moved around.

This is helpful when using exponents to solve [[equations]]. You also don't have to do straight calculation and can separate the numbers.

- Additionally, this rule can be used backwards $(x*y)^n=x^n*y^n$
