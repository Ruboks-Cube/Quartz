Values are the second part of the [[key value pair\|key value pairs]]
	Accessed with the [[key in python dictionaries\|key]] 

The values can be any [[Datatype]]:
```python
thisdict = {  
  "brand": "Ford",  # string [^4]
  "electric": False,  # bool [^3]
  "year": 1964,  # int [^2]
  "colors": ["red", "white", "blue"]  #list [^1]
}
```


[^1]: [[lists in python\|List]]

[^2]: [[int]]

[^3]: [[python bool\|bool]]

[^4]: [[String]]
