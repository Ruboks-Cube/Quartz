- Set items are [[Unordered meaning python\|unordered]] 
	- You can’t loop through them using an [[indexing\|index]] or a [[key in python dictionaries\|key]]
- You can loop through set items randomly using a [[for loop in python\|for loop]]
```python
# loop through a set and print the values
my_set = {"happy", "little", "trees"}
for x in my_set:
	print(x) # output: the items in random order
```
