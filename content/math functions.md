# Functions in Math
A function in math is something with two [[variable math]]s, one for an [[Input]] and one for a [[Output]]. More variables are possible. y=mx+b is an example of a function, it takes x, manipulates it, and that rule makes y. 

It's very similar to functions in coding.

If you have a graph, you can make sure the function is a function using the [[vertical line test]] 


[[Linear Equation]]s are an example of a function. They use basic [[arithmetic]] to manipulate variables.

- [[Domain]] is all possible $x$ values of a function
	- [[Domain\|finding the domain]] 
- [[Range]] is all possible $y$ values of a function
