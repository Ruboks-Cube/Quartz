#math/algebra/inequality/equations, [[One variable linear equation]], [[arithmetic]].

Very basic linear equations are linear equations that you can solve very easily. They deal with an [[variables as unknown values\|Unknown value]]/[[variable math\|variable]] that is very easy to figure out through basic [[arithmetic]].

- They can also be classified as ones where even if you did NOT know what [[Inverse Operation]] was, you would be able to solve within seconds

See [[One variable linear equation#A very basic linear equation\|A very basic linear equation]] for an example


Very basic linear equations can be solved through [[inspection]] (logic)
