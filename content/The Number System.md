#math , [[Numbers]] #math/numbertheory 
- Which letters stand for certain numbers:
	- $C$ → Complex
	- $R$ → Real
		- Imaginary does not have one but it’s prolly $i$
	- $Q$ → Rational, $\overline {Q}$ irrational
	- $Z$ → Integers
	- $N$ → Natural
- You know that [[Real Numbers]] and [[Imaginary Numbers]] exist
	- [[Rational Number\|Rational]] and [[Irrational Number\|irrational]]
	- [[integer\|integers]], [[Positive Numbers]], [[Negative Number\|Negative numbers]], [[Even]] and [[odd]] 
- These are all part of the number system, which we’ll go over


- Consider this set of numbers ${1,2,3,4,5,\dots}$ 
	- If we pick any 2 numbers from this set, like 4 and 6 when they are added we get another number in this set $4+6=10$
	- Multiplying any 2 numbers, like $4*6=24$ gives us another number in this set
	- What happens if we divide $2$ numbers, does $\frac{6}{4}$ belong in this set? Why not?
- This set ${1,2,3,4\dots}$ is considered to be [[Natural Numbers]] → [[Positive Numbers\|positive]]-[[integer\|integers]]
	- What happens if we subtract 2 numbers?
		- What happens if we have $4-6=-2$ or $\frac{6}{2}$?
		- Because of questions like these it was necessary to expand the number system
- The **set of integers** is defined as ${\dots,-3,-2,-1,0,1,2,3,\dots }$  $\to Z$ we can call this set $Z$ and $N \subset Z$ → $N$ is included in $Z$
	- The set of [[integer\|integers]] includes the set of [[Natural Numbers]]
- We still have to deal with [[Fraction\|fractions]]
	- For this the set of $Z$ was extended to the set of [[Rational Number\|rational numbers]], where $$
\begin{cases}
\frac{m}{n}:m\in Z,n\in Z, n\not\in  \\

\end{cases}
$$
	- Note that for this the division of [[zero, 0\|zero]] is not allowed
	- We now have the relationship of $N \subset Z \subset Q$ 
- Everything seems fine until we try to solve a problem like $x^2=2$ or determine the area of a [[Circle]] $\pi r^2$ 
	- Numbers from these problems could not be found in any of the sets found so far
	- This is where [[Irrational Number\|irrational numbers]] come into play
		- Irrational numbers become the set of numbers that did not belong to the set of $Q$ 
		- This set is denoted with $\overline {Q}$ , numbers that belong in this set are $\pi,e, \sqrt 2$ and all numbers that cannot be expressed with a [[Fraction]]
	- The [[Rational Number\|rational]] set and [[Irrational Number\|irrational]] set make up the set of [[Real Numbers]], that is $Q \cup \overline Q=R$ 
	- Real and [[Imaginary Numbers]] make up [[Complex Numbers]]
