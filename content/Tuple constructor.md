We can use `tuple()` to create a [[tuple]] in [[Python]] 

> [!Example]
> ```python
> my_tuple = tuple(("hello", "world", "yippee")) # two round brackets
> print(my_tuple) #output: ('hello', 'world', 'yippee')
> ```
