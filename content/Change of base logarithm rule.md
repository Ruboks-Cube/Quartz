#math/arithmetic/logarithm 

- For any time the $b\neq 1$ and is $>0$ we have $\log_{b}N=\frac{{\log_{c}N}}{\log_{c}b}$ where $c$ can be ANY number.
	- [[log by itself]] will be used in [[Using Ti-84+ calculator\|calculators]] because you can’t change the base from 10 for log and from $e$ for [[Natural Log]]. 
- Why???
	1. Start with $\log_{a}x=y \space \to \space a^y=x$  (they are [[equivalent]])  → There’s [[Substituting value of log for expression that equals the log]]
	2. We know that $\log_{b}x=\log_{b}a^y$ ([[substitution]]) 
	3. This becomes $\log_{b}x=y(\log_{b}a)$ through [[power logarithm rule]] 
	4. Divide the logs to get $y=\frac{{\log_{b}x}}{\log_{b}a}$ 


> [!info]
> We start with $[a\to x]=y$. This means that $a^y=x$ so from [[substitution]] since $a^y=x$, $b$ can be anything in $\log_{b}a^y=\log_{b}x$ 
> Then through [[power logarithm rule]] we get $y(\log_{b}a)=\log_{b}x$.
> Divide both sides by $(\log_{b}a)$ to get $y=\frac{{\log_{b}x}}{\log_{b}a}$ and $b$ can be ANYTHING YOU WANT.
