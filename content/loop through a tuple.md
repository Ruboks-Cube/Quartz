You can [[iteration control structure\|loop]] through [[tuple\|tuples]] like you can [[loop lists]]
using a [[for loop]]:
```python
my_tuple = ("banana", "yippee", "kai", "yo")
for i in my_tuple:
	print(i) # output: "banana", "yippee", "kai", "yo"
```

We can use [[len()]] in a [[for loop]] and a [[while loop]] as well
