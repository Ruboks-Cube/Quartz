We can use the [[keyword 'in' in python\|in]] and [['not In' in python\|not in]] keyword to check a [[Python Dictionaries\|Dictionary]] like [[keyword 'in' in python\|checking any other sequence]]
	We can use it to check if a [[key in python dictionaries\|key]] exists in a [[Python Dictionaries\|Dictionary]]
```python
thisdict = {  
  "brand": "Ford",  
  "model": "Mustang",  
  "year": 1964  
}  
if "model" in thisdict:  
  print("Yes, 'model' is one of the keys in the thisdict dictionary")
```
