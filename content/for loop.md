#Coding 

A for loop is a type of [[iteration control structure]] that repeats a certain code *for* each item in a [[Sequence in coding\|sequence]]

this is an example of a for loop:
```python
numbers = [1, 2, 3]
for n in numbers:
    print(n)
```
In this case "n" is the [[loop variable]]
	n is declared in line 2 `for n in numbers:`
	This variable can have any name
What the for loop does is it takes the [[loop variable]] and assigns it to each [[indexing\|index]] in a sequence, execute the code of the loop, and then move on to the next [[indexing\|index]]
