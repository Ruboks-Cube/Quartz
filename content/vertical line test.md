#math/graphing #math/function , [[math functions]]

- The vertical line test is to prove if a graph represents a function or not
	- for a graph to be considered a function, x values must correlate to 1 y value and 1 y value only
	- This means that for any x value, there should only be 1 y value
	- If we “scan” the graph with a vertical line, if at any point of the graph we see that the line touches 2 different points, the graph does not represent a function
