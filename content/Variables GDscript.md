#gamedevelopment/Godot #Coding/GDscript


- Like [[variable in coding\|variables]] in any other [[Coding Language]]
	- They [[keyword]] is `var` 
	- [[member variables GDscript]]

- To define a variable you put `var` before the [[variable identifier]], then “=” whatever value it is 
- In GDscript you don’t need the [[Datatype]] of the variable infront of the variable when you define it
	- While this makes it easier, it also means that the variable can be set to any [[Datatype]] at any point
	- I could turn an [[int]] variable into a [[String]] variable if I really wanted to
-  Variables in GDscript can also do [[Assignment Operators]], like in C#
