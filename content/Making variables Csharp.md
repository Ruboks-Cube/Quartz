# Basic Variable knowledge

## Variable rules
### Variable datatype
Variables must have their [[Datatype]] infront of the [[variable identifier]]]]

### Naming variables
The names of a variable are called [[variable identifier]] (identifiers)
- Names can contain letters, digits and the underscore character (_)
- Names must begin with a letter or underscore
- Names should start with a lowercase letter 
- it cannot contain whitespace
- Names are case sensitive ("myVar" and "myvar" are different variables)
- Reserved words (like C# keywords, such as `int` or `double`) cannot be used as names

## Why do we use variables?
A good question, variables are used to deal with possible values, values that you know are possible but can't be sure of when someone is running the program. Like user input. There are billions of ways someone could respond to Hello There! So it seems appropriate we add a value to it!
