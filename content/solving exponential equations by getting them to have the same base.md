#math/function/exponential 


- You can solve [[solving exponential equations\|exponential equations]] by getting them to have the same [[base]]
	- If you have a situation like $a^{x-y}=a^{z-x}$ then you know that $x-y=z-x$ because $a$ raised to $x-y$ is the same as $a$ raised to $z-x$
	- ! If the **SAME BASE** is raised to the **DIFFERENT** THINGS and is still **EQUAL** then maybe those “DIFFERENT THINGS” **arent DIFFERENT**
