#math/algebra #math/algebra/inequality/equations #math/rule [[equations\|Equation]], [[Expression]]

Combining Like terms refers to when you are adding a term with a variable like: $3x+4x$, in this one we can combine $3x$ and $4x$ to make $7x$. We can do that because if we [[Factoring\|factor out]] the $x$ we get $(3+4)x=7x$.

Another example of combining like terms would happen with [[Exponents]]. Take this: $2x^2+3x$
do you think that we can combine $2x^2+3x$? We cannot, because if we factor out $x$ in this case it will be $(2x+3)x$, because we only factored out one $x$. However, if we have something like $2x^2+3x^2$ we would get $5x^2$ since we factor out both.


> [!warning] Be careful when combining like terms when there are exponents
> When there are exponents in an expression, be careful when combining like terms! You can only combine like terms if they have the same value
