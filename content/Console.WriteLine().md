#Coding/Csharp 

The Console.WriteLine() [[function]] in [[Coding Csharp]] that takes a [[String]] (or other value) and simply writes it on the [[Console]]. It's an example of a [[Built In Function]], meaning it does not have to be defined by the user in order for it to work. The [[parameter]] is what we want to appear on the [[Console]]

The function simply takes the value and writes it without having the manipulate it except if [[Math Operations]] are used, for example:
```csharp
Console.WriteLine(3 + 3);
```

This function is similar to the [[Print()]] function in [[Python]] or [[Roblox Studio Lua]]



Similar to [[Console.Write]] with the only difference being that Console.Write does not form a new line to do the output.
