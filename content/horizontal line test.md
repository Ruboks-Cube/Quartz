#math/function #math/graphing , [[math functions]], [[Inverse Functions]]

- If you have a graph, to check if that graph represents a [[one to one function]] you can use the horizontal line test and [[vertical line test]]
	- The vertical line test is to see whether or not the original is a function (as you know)
	- Since the $x$ and $y$ values are flipped for the inverse, the horizontal line test will check if the inverse represents a function
	- like the vertical line test if any 2 points are touched by the horizontal line the inverse is not a function
- [[Parabola\|Parabola’s]] are never [[one to one function\|one to one]] because of this
