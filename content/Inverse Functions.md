#math/function [[function]], [[math functions]]

- Every [[function]] in math relates the [[Domain]] to the [[Range]].
	- Functions where both the inverse and the original are both functions are called [[one to one function\|one to one functions]] 
	- Inverse functions have flipped x and y values  
	- The original function to the power of [[Negative Number\|Negative]] 1 is how you define an inverse function so $f^{-1}(x)$ is inverse of $f(x)$
	- If you put the inverse inside the original (or the other way around) so like $f(f^{-1}(x))$ these 2 functions cancel each other out so you’re left w js $x$
## HOW TO FIND THE INVERSE OF A FUNCTION
- Lets say you’re trying to find the inverse of $f(x)=x$ 
	- first replace $f(x)$ with $y$ so we have $y=x$
	- Then switch the $x$ and $y$ so we have $x=y$
	- Then replace $y$ with $f^{-1}(x)$ so we have $f^{-1}(x)=x$ (flipped it around)
- That was js an example, here are steps for all!!!
	- Start with your function
	- replace function with y
	- Replace y with x and x with y (so they flip)
	- Solve for y
	- Replace y with function to the power of negative 1
- These steps effectively reverse the function
### Checking if 2 functions are inverse
- You can check is $f(x)$ and $g(x)$ are inverse with [[composition functions]]
	- We know the inverse and the original cancel each other out
	- If $f(g(x))$ AND (make sure to do both) $g(f(x))$ both equal $x$ then you have an inverse
	- In IB AAF class if it IS an inverse we write it as: $f(g(x))=g(f(x))=x$


## Making inverses stay functions with amputations 
- You can amputate functions to make sure that the inverse stays as a function
	- What do I mean by this? You ask? 
- LETS START WITH A [[Parabola]]
	- $f(x)=x^2$, the inverse is $f^{-1}(x)=\sqrt{x}$. Switching the $y$ and $x$ values the parabola will js be sideways, does not pass the [[horizontal line test]] or [[vertical line test]]
