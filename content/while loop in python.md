The while [[iteration control structure\|loop]] is the other [[iteration control structure]] in [[Python]] along with the [[for loop]] 

- The while loop repeatedly executes what’s inside it as long as a [[condition in coding\|condition]] is true

```python
# basic syntax:
while <condition>:
	<code to be executed>
```

- [[Break statement]]
- [[Continue statement]]

- [[Else]] also works in while loops
