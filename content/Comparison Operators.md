#Coding 

- Comparison operators are used to compare two values ([[variable in coding\|Variables]]) using [[Inequalities]]
	- The [[Returning in code\|return]] value of a Comparison is a [[Bool]]

- Here is an example in C#
	```csharp
		int x = 5;
		int y = 3;
		Console.WriteLine(x > y); // returns True because 5 is greater than 3
	```
	- Another example is:
		```csharp
		int x = 5, y = 3;
		bool comparison = x>y;
		Console.WriteLine(x>y);
		```
