You can use a [[python bool\|bool]] with [[if statement python\|if statements]] 
	One of the most common ways to use bools actually
```python
a = 200
b = 33

if b > a:
  print("b is greater than a")
else:
  print("b is not greater than a") #this is what will print
```

Common to use with [[User Input]] as well
```python
input1 = int(input("Enter first number: ").strip()) 
input2 = int(input("Enter second number: ").strip()) 

if input1 > input2:
    print(f"{input1} is greater than {input2}")
elif input1 < input2:
        print(f"{input1} is less than {input2}")
else:
        print(f"{input1} is equal to {input2}")
# This is a simple Python program to compare two numbers and print the result.
```
