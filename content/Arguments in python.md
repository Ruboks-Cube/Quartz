This note mostly talks about the difference between arguments and [[parameters in python]]

In [[Python]], from a [[Functions in coding\|function]] perspective both built in and [[Python custom functions\|custom]]:
	Parameters are the variables defined in a function
	Arguments are the values sent to the variable when the function is [[calling functions\|called]]
