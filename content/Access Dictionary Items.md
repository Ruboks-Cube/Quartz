You can access a [[value in dictionary\|Value]] in a [[Python Dictionaries\|Dictionary]] by referring to the [[key in python dictionaries\|key]]
```python
thisdict = {  
  "brand": "Ford",  
  "model": "Mustang",  
  "year": 1964  
}  
x = thisdict["model"]
print(x) #output: Mustang
```
- [[get()]] achieves the same thing
- [[keys()]]
- [[values()]]
- [[items()]]
