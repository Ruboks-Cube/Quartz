The code [[Indentation in Python\|indented]] in the “Else” [[keyword]] is what will execute if none of the previous [[selection control structure\|conditions]] were true

You can put an [[Else python\|else]] at the end of a [[while loop in python]] which is what the code runs after the [[condition in coding\|condition]] becomes false

```python
i = 1  
while i < 6:  
  print(i)  
  i += 1  
else:  
  print("i is no longer less than 6")
```

This is true for [[if statement python\|if statements]] and [[while loop in python\|While loops]]

- With [[Python Try Except]], [[else in try except\|else is a bit different]] but it’s premise stays the same
	- It’s not a *condition* rather just something that executes when no [[python errors]] are raised
