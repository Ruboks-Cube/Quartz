#math #math/numbertheory , [[Numbers]], [[Number Line]], [[Negative Number]], [[Positive Numbers]], [[Cartesian coordinate system]]

- The distance a number is from 0. 
	- If the number is [[Positive Numbers\|Positive]] then we just take that number, so if the number is 3, the absolute value is also 3
	- If the number is [[Negative Number\|negative]], the distance is not $-3$ because distance can't really be negative! We take the $-3$, and since it's 3 steps away from 0 on the numberline

> [!check] idea
> - Absolute value is always positive
> - We sometimes call the absolute value the **magnitude** of a number

- We write absolute value with $||$, like $|3|=3$, as in the absolute value of $3$ is $3$, while $|-3|=3$ the absolute value of $-3$ is $3$.

- Absolute value is a [[math functions\|function]], so it can be graphed! → [[graphing functions]]
