A Trapezoid is a [[2 dimensional]] [[quadrilateral]] that has 1 set of [[parallel]] lines.

### Area of a Trapezoid
How do we find the area of a trapezoid?
![[Files/Images/Trapezoid 2024-03-06 19.57.05.excalidraw]]
We flip the parallelogram Horizontally and Vertically and combine it to the possible locations that we can. 
$\overline {AB} \parallel \overline{CD}$ which means that $\angle B+\angle D=180$ and $\angle A + \angle C=180$. If you don't know why, try visualizing it so the sides that are not parallel are [[Angles and parallel lines\|transversal lines]].
Using the Corresponding sides we find that if we combine it so the $\overline {BD}$ sides are combined it works, and if we make it so the $\overline {AC}$ sides are combined it also works.
![[Files/Images/Trapezoid 2024-03-06 20.05.16.excalidraw]]
In this diagram we can see that when we combine the horizontally and vertically flipped trapezoid it works! It makes a [[parallelogram]], and we know how to find the area of a parallelogram.

![[Files/Images/Trapezoid 2024-03-06 20.09.31.excalidraw]]
The area of a trapezoid is $\frac{hb}{2}$ where h is the height and b is the base of the whole parallelogram.
