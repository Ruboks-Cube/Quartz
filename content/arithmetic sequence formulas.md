#math/algebra/sequence 

- An Arithmetic sequence is a sequence that has a common difference between every number in the sequence → like a [[Linear Equation]]
- We are used to describing [[arithmetic]]-[[Sequence\|sequences]] like this:
	- $3,5,7\dots$ 
- But we can also use a [[formula]] to describe the sequence.
	- There are both [[Recursive Functions in arithmetic\|Recursive formulas]] and [[explicit functions in arithmetic\|explicit formulas]] 
- To remain general, formulas use $n$ to represent any [[term]] number and $a(n)$ to represent the $n^{th}$ term of the sequence
	- they make a [[function]]
- For any term number $n$, $a(n-1)$ represents the place of the number in the [[Sequence]] before $n$.  → We are subtracting by $1$
