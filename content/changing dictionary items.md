You can change the [[value in dictionary\|Value]] of an item in a [[Python Dictionaries\|Dictionary]] by referring to the [[key in python dictionaries\|key]]
```python
thisdict = {  
  "brand": "Ford",  
  "model": "Mustang",  
  "year": 1964  
}  
thisdict["year"] = 2018
```
