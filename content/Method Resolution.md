Method resolution is how the [[Coding\|program]] determines which *version* of a [[Method]] to use
	This is apparent in [[object oriented programming]] with [[python class\|classes]]/custom datatypes
- Will the program [[method overriding\|override]] the [[parent class]] and use the [[child class]] method? Or will it go to the parent class and use the [[Method]] there
	- This is how [[python inheritance]] works
