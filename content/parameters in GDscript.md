#Coding #gamedevelopment/Godot 

- Parameters in GD script are really similar to [[Parameters in Lua]]
- You put it in the parenthesis in [[Creating custom functions in Godot\|the function you make]] 
	```GDscript
	func draw_square(size):
	```
	- In this case a parameter is “size”
- You can also have multiple parameters
```GDscript
func draw_rectangle(length, width)
```
