#math/algebra/inequality/equations, #math/algebra , [[algebra]], [[variable math\|variable]], #problemsolving 

In algebra, Variables are almost always unknown values that we have to figure out. Like in [[One variable linear equation\|One variable linear equations]].

In word problems, when we have unknown values, immediately make a variable for it so you actually can work with the value instead of being like "Huh?"
