#math/geometry -> [[congruent]]


- Having the same [[2D shape\|shape]] but not necessarily the same [[size]] which makes it different from [[congruent]]
	- The [[corresponding angles\|corresponding]]--[[Angles]] will be the same (aka [[congruent]]) 
- 2 shapes are said to be similar if there are a set of [[Rigid Transformations]] and [[Dilation\|Dilations]] that will map one shape onto the other
	- [[similarity transformations]]
		- When using transformations to test for similarity we can first try to examine the dilation.
		- If all [[corresponding sides]] are [[Proportion\|proportional]] meaning they have the same [[Ratio]] then the ratio of the sides is the [[Scale Factor]] of the dilation. 
		- Then you can see whether [[Rotation\|rotations]] or [[Reflection\|reflections]] will allow you to get the right [[orientation]]
		- A [[Translation]] may be needed to map the shapes on one another
