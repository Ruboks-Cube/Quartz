The & operator is like the [[intersection()]] method except it doesn’t allow you to intersect with different [[Datatype\|datatypes]]

```python
set1 = {"apple", "banana", "cherry"}  
set2 = {"google", "microsoft", "apple"}  
  
set3 = set1 & set2  
print(set3) # output {'apple'}  
```
