#math/algebra/inequality/equations/polynomial , [[quadratic equations]] #math/algebra/inequality/equations/polynomial/quadratic 

- A way of solving [[quadratic equations]].

- Completing the square basically turns the [[Standard Form quadratic]] of a [[quadratic equations\|Quadratic]] into [[Vertex Form]]
- We start with a normal standard form quadratic, $Ax^2+Bx+C=0$
	- $Ax^2+Bx+C=0$, the terms $Ax^2+Bx$ can be part of a [[Perfect Squares\|Square]], meaning we can have something like this $(x+?)^2$ where we have $2$ [[term\|Terms]] being squared.
	- The [[quadratic formula]] is actually derived from [[completing the square]] so you're good if you know how it works
	- $Ax^2+Bx+C=0$
		- We first divide both sides of the equation by $A$ to get rid of the [[Coefficient]].
			- $x^2+\frac{B}{A}x+\frac{C}{A}=0$  
		- We can add ${} (\frac{B}{2A})^2 {}$ to both sides of the equation.
			- ? Why? Because when we have an [[Expression]] like $(x+B)^2$ see what happens when we expand it
				- $x^2+2Bx+B^2$. See how the second term is $2Bx$ and the last term is $B^2$ which is $\frac{1}{2}$ the second term [[Perfect Squares\|Squared]]??? This is true for all situations (we js proved it by using [[variable math\|variables]])
		- After adding ${} \left( \frac{B}{2A} \right)^2 {}$ to both sides we have: $x^2+\frac{B}{A}x+\left( \frac{B}{2A} \right)^2=-\frac{C}{A}+\left( \frac{B}{2A} \right)^2$ 
			- (we subtracted $\frac{C}{A}$ so it's on the other side now)
		- We can now complete the square so the left side is a completed square $\left( x+\frac{B}{2A} \right)^2=-\frac{C}{A}+\left( \frac{B}{2A} \right)^2$ 
		- This is almost [[Vertex Form]], I hope you can see why cause I'm not going farther. We've completed the square.
