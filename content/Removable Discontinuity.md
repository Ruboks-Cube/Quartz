#math/function/rational 


- A [[point]] on the [[Cartesian coordinate system\|graph]] that is [[undefined]] or doesn’t fit the rest of the graph
	- How do you find this?
		- If 2 [[Factoring\|factors]] are able to be canceled from the [[Rational Functions]] like this:
			$\frac{(x-7)(x+3)}{(x-12)(x+3)}$ 
		- The $x$ value of the [[ordered pair]] is whatever makes them both 0
			- In this case it is $-3$
		- The $y$ value is what it is plugged into the *simplified* [[equations\|Equation]]:
			$\frac{-3-7}{-3-12}=\frac{2}{3}$ 
		- We plugged in $-3$ into $\frac{x-7}{x-12}$
	- Additionally, if there is a [[Removable Discontinuity]], you use the simplified equation for the rest of the graphing process like finding the [[Asymptote\|Asymptotes]]
	- This means there isn’t a [[Finding Vertical Asymptote of Rational Functions\|Vertical Asymptote]] $x=-3$ in this equation
