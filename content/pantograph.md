Summary
The pantograph is a copying image that can be used to enlarge or shrink images. It works by having 2 pairs of parallel lines that intersect to make a parallelogram. 3 points are then created for the use of the pantograph. There is the fixed point, tracer, and the drawing point. The fixed point stays in the same place throughout the copying, the tracer is the point that traces the original image, and the pencil (or pen) is what produces the scaled copy. By changing the distance between the fixed point and the tracer point and the tracer point and the drawing point you can change the scale of the image. This is similar to dilations because it IS a dilation, you are dilating a shape from the fixed point and the new shape is similar to the original. We can use similar triangles to show how the pantograph works because if we took an equilateral triangle and made a copy of it using a pantograph the copy would have the same angles since they're similar.
