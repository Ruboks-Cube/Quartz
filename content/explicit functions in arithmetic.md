#math/algebra/sequence #math/function 

- Here is an explicit formula of $3,5,7\dots$
	- $a(n)=3+2(n-1)$
	- This formula allows us to simply [[Plugging in variables\|plug in]] the number of the term we want to get the value of that term 
	- We have the starting value $3$ and how many jumps between $n$ and the starting value is $(n-1)$ multiplied by 2
	- This reminds me of js a normal [[Linear Equation]] 
- This is the structure of an explicit formula: $a(n)=A+B(n-1)$ → $A$ is the starting number and $B$ is the common difference between adjacent numbers in the sequence
	- The structure REALLY reminds me of [[slope intercept form]] → The difference between the 2 is that we multiply $B$ (the common difference) by $n-1$ since the starting value of a [[Recursive Functions in arithmetic\|recursive formula]] and an [[explicit functions in arithmetic\|explicit formula]] is $a(1)$ but for [[slope intercept form]] we consider it to be $b$ (the [[intercept\|y intercept]])
