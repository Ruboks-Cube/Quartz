When you use an [[Arbitrary Arguments python\|Arbitrary argument]] you are creating a [[tuple]] 

If you’re using a default function with [[Default number of arguments]], the function isn’t expecting a [[tuple]], it’s just expecting values that it can match to [[variable in coding\|variables]]. 

This is why we get a [[TypeError]]
