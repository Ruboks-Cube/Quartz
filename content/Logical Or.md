#Coding/Csharp 

- Returns true if at least one of the statements is true
	- “||”
	- `x<5||x<4`
```csharp
int x = 5;
Console.WriteLine(x>3 || x<4);
```
- This returns true because $x>3$ so ONE of them is true 
```csharp
int x = 5;
Console.WriteLine(x>3 || x<4);
```
