A set is a [[collections in python\|type of collection]] in [[Python]]
	A set is unordered, unchangeable, and [[indexing\|unindexed]]
	Written with {}
```python
my_set = {"happy", "little", "trees"}
print(my_set) # output may be in any order, e.g. {'little', 'happy', 'trees'}
```

- sets are [[immutable]], but you can add and remove set items
