#math/algebra/inequality/equations , [[equations\|Equation]], [[Linear Equation]]

- A 2 variable linear equation is just a [[Linear Equation]] with 2 variables, it's a very fitting name
- No more than 1 variable per term
- highest exponent for a variable is 1

- Here is an example: 
	- $2x=y$

- Here are non examples:
	- $2x^2=y$
	- $xy=2$
	- $\frac{a}{b}+\frac{b}{a}=3$

# Problems
## Introduction Problem (tables) and [[ordered pair\|Ordered Pairs]]
![[Files/Images/Pasted image 20240427090953.png]]
- This is an example of a table, it represents a [[math functions\|Function]] 
- The value of one variable changes depending on the value of another variable

- We can finish the table by [[Plugging in variables\|Plugging in the variable]] thats already there which turns this into a simple [[One variable linear equation]].
- (The missing value is bolded and italicized)

| x        | y       | Linear equation |

- The last 2 we just do the same thing, but it's decimals.

- These are all possible solutions to the equation, which we can write as ordered pairs, for example $(5,1)$ is a [[ordered pair]]

## Introduction continued

- Additionally, we can create a formula by isolating x and [[solving multi variable equations in terms of another variable]]
- Our original equation was $2x-3y=7$ 
	- Add $3y$ to both sides to get $2x=7+3y$
	- Divide both sides by $2$ to get $x=\frac{{7+3y}}{2}$
- Now we have a formula, we can choose a value for y and the corresponding value of x will make an [[ordered pair]] that is a possible solution to the equation
- We can imagine $y$ as the [[independent variable]] and $x$ as the [[dependent variable]]
	- We can change the value of $y$ to get the value of $x$
	- This shows that there are an infinite amount of solutions since there are infinite values of $y$ which will result in infinite values of $x$

## What if we have 2 equations??
- We saw in the previous problem that there are an infinite amount of solutions for $x$ and $y$ when we have a single equation, but what about $2$ equations? Are there an infinite amount of solutions for $x$ and $y$ that satisfy both???
![[Files/Images/Pasted image 20240427093209.png]]
- We can first try finding an ordered pair that gives solution to the first equation
	- $x+3y=6$, when $x=0$, $y=2$
	- $\left( 1,{\frac{5}{3}} \right)$ is another solution
	- $\left( 2,{\frac{4}{3}} \right)$ 
	- $(3,1)$ or $\left( 3,{\frac{3}{3}} \right)$ is another one
	- $\left( 4, \frac{2}{3} \right)$, we see that whenever we increase $x$ by $1$ $y$ decreases by $\frac{1}{3}$

	- Now for the second equation $2x-5y=1$
		- When $x=0$, $y=-\frac{1}{5}$
		- When $x=1$ $y=\frac{1}{5}$
		- $\left( 2,{\frac{3}{5}} \right)$
		- $3, 1$ or $3,{\frac{5}{5}}$
		- $4,{\frac{7}{5}}$
- We see that $3,1$ is on both lists, which means that $3,1$ is a solution to both equations
- This is an example of a [[system of equations]]
> [!tip] Tip
> To get [[ordered pair\|Ordered Pairs]] you first input a value in one variable and solve it like a normal [[Linear Equation]]
