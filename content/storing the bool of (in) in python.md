#Coding/python 

We can use the [[using print() with (in) in python\|print() with in keyword]] to [[keyword 'in' in python\|check a sequence]]

We can also do [[using (in) in an if statement in python\|an if statement with in keyword]] to [[keyword 'in' in python\|check a sequence]]

We can amend these to simply store a [[variable in coding\|variable]] rather than [[Print()\|printing]] something directly onto the console

[[using print() with (in) in python\|print() with (in) keyword]] amending:
```python
a = "hello world"
b = "hello" in a
print(b) #prints true because hello is in a
```


[[using (in) in an if statement in python]] amending:
```python
a = "hello world"
if "hello" in a:
	b = True
print(b) #prints True
```
