#Coding 

To [[calling functions\|call a function]] you first have to declare it, unless it is built in.

## What is it?
The process of making a function and making it do something is called declaring a function.
When you declare a function, you usually have 4 main parts when you declare it.

1. The name of the function (self explanatory)
2. Any [[parameter]]s you might use
3. The block of code inside the function, what makes something an output, whether on the console or in a game in [[object oriented programming]]
4. end, some kind of end in the [[script]] so it knows that your done creating the function, if you don't put this, the script will think everything below it is you declaring a function. You certainly don't want this.
